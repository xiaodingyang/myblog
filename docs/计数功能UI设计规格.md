- 位置：页面标题右侧

- 切换时卡片数值有 `fade` + 微位移过渡（`opacity 0→1` + `translateY(4px→0)`，`300ms`）

## 四、区域 2：访问趋势图（Canvas）

### 4.1 容器

| 属性 | 值 |
|------|-----|
| 高度 | PC `360px`，移动端 `240px` |
| 背景 | 毛玻璃卡片（同数据卡片样式） |
| 内边距 | `24px`（图表区域留出坐标轴空间） |
| 标题 | 左上角「访问趋势」，`16px`，`#1e293b` |
| 切换按钮 | 右上角，`7天` / `30天`，样式同时间范围切换 |

### 4.2 高 DPI 适配

Canvas 必须根据 `window.devicePixelRatio` 进行高清渲染，确保 Retina 屏及其他高 DPI 设备下曲线、文字、数据点不模糊。

| 属性 | 说明 |
|------|------|
| 实际像素 | `canvas.width = cssWidth * devicePixelRatio`，`canvas.height = cssHeight * devicePixelRatio` |
| CSS 尺寸 | 保持不变（通过 `style.width` / `style.height` 设置） |
| 绘图缩放 | 绑定后调用 `ctx.scale(devicePixelRatio, devicePixelRatio)` |
| 监听变化 | 使用 `matchMedia('(resolution: Xdppx)')` 或 `resize` 事件监听 DPR 变化（如外接显示器切换），动态重绘 |

### 4.3 坐标系

| 属性 | 值 |
|------|-----|
| X 轴 | 日期标签，`12px`，`#94a3b8`，间距自适应（7天全显示，30天隔 5 天显示） |
| Y 轴 | 数值标签，`12px`，`#94a3b8`，5 条水平参考线 |
| 参考线 | `1px dashed rgba(148, 163, 184, 0.2)` |
| 图表区域 | 左留 `50px`（Y 轴标签），底留 `30px`（X 轴标签），上留 `20px`，右留 `20px` |

### 4.4 曲线

| 属性 | PV 曲线 | UV 曲线 |
|------|---------|--------|
| 颜色 | `var(--theme-primary)` | `var(--theme-primary)` 透明度 50% |
| 线宽 | `2px` | `2px` |
| 算法 | Catmull-Rom 样条插值 | 同左 |
| 发光（P1） | `shadowBlur: 20` + `shadowBlur: 10` + 实线三层叠加 | 同左，透明度降低 |

> 发光效果性能优化：建议使用离屏 Canvas 预渲染模糊线条叠加，或降级为单层 `shadowBlur: 10`。低端设备（`navigator.hardwareConcurrency <= 2`）可跳过发光效果。

### 4.5 面积填充

- PV 曲线下方：`linearGradient` 从 `rgba(主题色, 0.3)` → `rgba(主题色, 0)`
- UV 曲线下方：`linearGradient` 从 `rgba(主题色, 0.15)` → `rgba(主题色, 0)`

### 4.6 数据点

| 属性 | 值 |
|------|-----|
| 半径 | `4px`（静态，无常驻动画） |
| 填充 | 主题色 |
| 描边 | `2px white` |
| Hover 半径 | `6px` |
| Hover 涟漪 | 圆环从 `r=5` 扩散到 `r=30`，`opacity 0.6→0`，`600ms` |

> 不使用常驻呼吸脉冲动画，避免 30 天模式下 60 个数据点同时动画导致的 Canvas 重绘性能问题。仅保留 Hover 涟漪效果。

### 4.7 Tooltip

```
┌─────────────────┐
│  04-06 周日       │  ← 日期
│  PV  1,234       │  ← 主题色圆点 + 数值
│  UV    567       │  ← 淡色圆点 + 数值
└─────────────────┘
```

| 属性 | 值 |
|------|-----|
| 背景 | `rgba(255, 255, 255, 0.8)` + `backdrop-filter: blur(8px)` |
| 边框 | `1px solid rgba(255, 255, 255, 0.3)` |
| 圆角 | `8px` |
| 阴影 | `0 4px 12px rgba(0,0,0,0.1)` |
| 内边距 | `12px` |
| 文字 | 日期 `12px #64748b`，数值 `14px #1e293b font-weight: 500` |
| 跟随 | 鼠标位置，偏移 `(12px, -12px)`，不超出容器边界 |
| 触摸设备 | Tooltip 固定显示在图表顶部居中，手指按住时显示，抬起后延迟 `1.5s` 消失 |

### 4.8 图例

- 位置：标题右侧、切换按钮左侧
- 样式：`● PV  ● UV`，圆点 `8px`，文字 `12px #64748b`

### 4.9 动画时序

| 动画 | 触发 | 时长 | 缓动 |
|------|------|------|------|
| 入场画线 | 首次加载 | `1200ms` | `easeOutCubic` |
| 数据点弹出 | 画线经过时 | `300ms` | `easeOutBack` |
| Hover 涟漪 | 鼠标靠近 | `600ms` | `easeOut` |
| 切换过渡 | 7天↔30天 | `800ms` | 旧淡出 `200ms` + 新画入 `600ms` |
| 主题色过渡 | 切换主题 | `800ms` | `linear` 插值 |

### 6.1 布局

| 中心文字 | 总访问数，`24px bold #1e293b` + 「总访问」`12px #64748b` |

### 6.3 配色方案

基于主题色生成 6 色色板（HSL 色相旋转）：

| 序号 | 色相偏移 | 用途 |
|------|---------|------|
| 1 | `+0°` | 第 1 来源（主题色本身） |
| 2 | `+30°` | 第 2 来源 |
| 3 | `+60°` | 第 3 来源 |
| 4 | `+120°` | 第 4 来源 |
| 5 | `+180°` | 第 5 来源 |
| 6 | 灰色 `#cbd5e1` | 「其他」 |

> **色彩冲突风险提示**：部分主题色（如绿色 `#10b981`）旋转 `+30°`/`+60°` 后会产生黄绿色系，相邻扇区区分度可能较低。建议前端实现时：(1) 对生成的色板进行对比度校验（相邻颜色 ΔE > 30）；(2) 如对比度不足，调整色相偏移角度（如改为 `+0°/+45°/+90°/+150°/+210°`）或增加明度差异；(3) 低端设备可降级为预设色板（如 Ant Design Charts 默认色板）。

## 十、主题色适配

所有跟随主题色的元素：

- 数据卡片图标颜色
- 时间范围切换选中态背景
- 趋势图曲线 + 面积填充 + 数据点
- 热门页面前三名徽章背景
- 饼图色板（基于主题色 HSL 旋转）
- 搜索按钮背景
- 表格行 hover 背景
- 错误态重试按钮描边

切换主题色时，以上元素通过 CSS 变量 `var(--theme-primary)` 自动跟随，Canvas 部分通过 `colorModel` 监听 + `800ms` 插值过渡。

## 十一、深色模式适配（P1）

当前规格基于浅色模式设计。如需支持深色模式，建议采用以下适配方案：

### 11.1 色彩映射

| 元素 | 浅色模式 | 深色模式 |
|------|---------|---------|
| 页面背景 | `#ffffff` | `#0f172a`（Slate 900） |
| 卡片背景 | `rgba(255, 255, 255, 0.6)` | `rgba(30, 41, 59, 0.6)`（Slate 800 半透明） |
| 卡片边框 | `rgba(255, 255, 255, 0.3)` | `rgba(148, 163, 184, 0.2)`（Slate 400 低透明） |
| 主标题文字 | `#1e293b`（Slate 800） | `#f1f5f9`（Slate 100） |
| 正文文字 | `#334155`（Slate 700） | `#cbd5e1`（Slate 300） |
| 辅助文字 | `#64748b`（Slate 500） | `#94a3b8`（Slate 400） |
| 参考线 | `rgba(148, 163, 184, 0.2)` | `rgba(148, 163, 184, 0.1)` |
| 表格斑马纹 | `rgba(0, 0, 0, 0.02)` | `rgba(255, 255, 255, 0.02)` |
| 表格 hover | `rgba(主题色, 0.05)` | `rgba(主题色, 0.1)` |

### 11.2 实现方式

- 使用 CSS 变量 + `[data-theme="dark"]` 选择器切换色值
- 或使用 `@media (prefers-color-scheme: dark)` 自动跟随系统
- Canvas 部分需监听主题切换事件，重新绘制时使用对应色值

### 11.3 注意事项

- 主题色在深色模式下可能需要提高亮度（Lightness +10%~15%），确保对比度符合 WCAG AA 标准（4.5:1）
- 玻璃拟态效果在深色背景下需调整模糊半径和透明度，避免过于暗淡
- 饼图色板在深色模式下建议提高饱和度（Saturation +10%），增强视觉区分度

> 深色模式为 P1 优化项，不阻塞 P0 开发。建议在浅色模式稳定后再实现。
