| 文字色 | 主标题 `#1e293b` / 正文 `#334155` / 辅助 `#64748b` | Slate 色系 |

## 三、区域 1：数据卡片行

### 3.1 布局

- PC：4 列等宽，`gap: 16px`，使用 CSS Grid `grid-template-columns: repeat(4, 1fr)`
- 平板（≤1024px）：2×2 网格
- 移动端（≤768px）：单列堆叠

### 3.2 单卡片规格

```
┌──────────────────────────┐
│  📊 图标    今日 PV       │  ← 标签行
│                          │
│       1,234              │  ← 数值（大字）
│                          │
│  较昨日 ↑ 12.5%          │  ← 趋势（可选，P1）
└──────────────────────────┘
```

| 属性 | 值 |
|------|-----|
| 卡片背景 | `rgba(255, 255, 255, 0.6)` |
| 模糊 | `backdrop-filter: blur(12px)` |
| 边框 | `1px solid rgba(255, 255, 255, 0.3)` |
| 圆角 | `12px` |
| 内边距 | `20px` |
| 阴影 | `0 4px 6px -1px rgba(0,0,0,0.1)` |
| 图标 | Ant Design 图标，颜色跟随主题色，大小 `20px` |
| 标签文字 | `14px`，颜色 `#64748b` |
| 数值文字 | `32px`，`font-weight: 600`，颜色 `#1e293b` |
| 趋势文字 | `12px`，上涨 `#10b981`，下降 `#ef4444` |

### 3.3 时间范围切换

- 位置：页面标题右侧
- 组件：Ant Design `Segmented`（分段控制器）
- 选项：今日 / 昨日 / 本周 / 本月
- 选中态：主题色背景 + 白色文字
- 未选中态：透明背景 + `#64748b` 文字
- 切换时卡片数值有 `fade` 过渡（`opacity 0→1`，`300ms`）

### 3.4 四张卡片内容

| 卡片 | 图标 | 标签 | 数据来源 |
|------|------|------|----------|
| 1 | `EyeOutlined` | PV（页面浏览） | overview.pv |
| 2 | `UserOutlined` | UV（独立访客） | overview.uv |
| 3 | `ClockCircleOutlined` | 平均停留 | overview.avgDuration（格式化为 `Xm Ys`） |
| 4 | `RiseOutlined` | PV 趋势 | 当前范围 vs 上一周期的变化率（P1） |

> P0 阶段第 4 张卡片可先显示昨日 UV，P1 再改为趋势对比。

## 四、区域 2：访问趋势图（Canvas）

### 4.1 容器

| 属性 | 值 |
|------|-----|
| 高度 | PC `360px`，移动端 `240px` |
| 背景 | 毛玻璃卡片（同数据卡片样式） |
| 内边距 | `24px`（图表区域留出坐标轴空间） |
| 标题 | 左上角「访问趋势」，`16px`，`#1e293b` |
| 切换按钮 | 右上角，`7天` / `30天`，样式同时间范围切换 |

### 4.2 坐标系

| 属性 | 值 |
|------|-----|
| X 轴 | 日期标签，`12px`，`#94a3b8`，间距自适应（7天全显示，30天隔 5 天显示） |
| Y 轴 | 数值标签，`12px`，`#94a3b8`，5 条水平参考线 |
| 参考线 | `1px dashed rgba(148, 163, 184, 0.2)` |
| 图表区域 | 左留 `50px`（Y 轴标签），底留 `30px`（X 轴标签），上留 `20px`，右留 `20px` |

### 4.3 曲线

| 属性 | PV 曲线 | UV 曲线 |
|------|---------|--------|
| 颜色 | `var(--theme-primary)` | `var(--theme-primary)` 透明度 50% |
| 线宽 | `2px` | `2px` |
| 算法 | Catmull-Rom 样条插值 | 同左 |
| 发光（P1） | `shadowBlur: 20` + `shadowBlur: 10` + 实线三层叠加 | 同左，透明度降低 |

### 4.4 面积填充

- PV 曲线下方：`linearGradient` 从 `rgba(主题色, 0.3)` → `rgba(主题色, 0)`
- UV 曲线下方：`linearGradient` 从 `rgba(主题色, 0.15)` → `rgba(主题色, 0)`

### 4.5 数据点

| 属性 | 值 |
|------|-----|
| 半径 | `4px`（呼吸脉冲 `4px↔5px`，周期 `2s`） |
| 填充 | 主题色 |
| 描边 | `2px white` |
| Hover 半径 | `6px` |
| Hover 涟漪 | 圆环从 `r=5` 扩散到 `r=30`，`opacity 0.6→0`，`600ms` |

### 4.6 Tooltip

```
┌─────────────────┐
│  04-06 周日       │  ← 日期
│  PV  1,234       │  ← 主题色圆点 + 数值
│  UV    567       │  ← 淡色圆点 + 数值
└─────────────────┘
```

| 属性 | 值 |
|------|-----|
| 背景 | `rgba(255, 255, 255, 0.8)` + `backdrop-filter: blur(8px)` |
| 边框 | `1px solid rgba(255, 255, 255, 0.3)` |
| 圆角 | `8px` |
| 阴影 | `0 4px 12px rgba(0,0,0,0.1)` |
| 内边距 | `12px` |
| 文字 | 日期 `12px #64748b`，数值 `14px #1e293b font-weight: 500` |
| 跟随 | 鼠标位置，偏移 `(12px, -12px)`，不超出容器边界 |

### 4.7 图例

- 位置：标题右侧、切换按钮左侧
- 样式：`● PV  ● UV`，圆点 `8px`，文字 `12px #64748b`

### 4.8 动画时序

| 动画 | 触发 | 时长 | 缓动 |
|------|------|------|------|
| 入场画线 | 首次加载 | `1200ms` | `easeOutCubic` |
| 数据点弹出 | 画线经过时 | `300ms` | `easeOutBack` |
| 呼吸脉冲 | 常驻 | `2000ms` 循环 | `sine` |
| Hover 涟漪 | 鼠标靠近 | `600ms` | `easeOut` |
| 切换过渡 | 7天↔30天 | `800ms` | 旧淡出 `400ms` + 新画入 `800ms` |
| 主题色过渡 | 切换主题 | `800ms` | `linear` 插值 |

| 3 | `+60°` | 第 3 来源 |
| 4 | `+120°` | 第 4 来源 |
| 5 | `+180°` | 第 5 来源 |
| 6 | 灰色 `#cbd5e1` | 「其他」 |

## 十、主题色适配

所有跟随主题色的元素：

- 数据卡片图标颜色
- 时间范围切换选中态背景
- 趋势图曲线 + 面积填充 + 数据点
- 热门页面前三名徽章背景
- 饼图色板（基于主题色 HSL 旋转）
- 搜索按钮背景
- 表格行 hover 背景
- 错误态重试按钮描边

切换主题色时，以上元素通过 CSS 变量 `var(--theme-primary)` 自动跟随，Canvas 部分通过 `colorModel` 监听 + `800ms` 插值过渡。
