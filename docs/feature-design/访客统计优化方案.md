# 访客统计功能优化方案

## 当前问题

### 1. 时区处理过于复杂
`visitController.js` 中手动计算北京时间偏移（`BJ_MS = 8 * 3600000`），代码重复且容易出错。

### 2. IP 隐私保护不足
后端存储完整 IP 地址，前端仅显示时脱敏。数据库泄露会暴露用户隐私。

### 3. 前端防抖逻辑有缺陷
`lastVisitTime` 存在内存中，页面刷新后会丢失，导致同一页面刷新后可能不计数。

### 4. MongoDB 索引不够精准
`{ createdAt: -1 }` 降序索引不适合 `getTrend` 的范围查询（`$gte`）。

### 5. 前端错误处理粗糙
所有接口失败都显示"数据加载失败"，无法区分网络错误、权限错误等。

### 1. 时区处理优化

**创建时区工具函数**：
```javascript
// backend/src/utils/dateHelper.js
const dayjs = require('dayjs');
const utc = require('dayjs/plugin/utc');
const timezone = require('dayjs/plugin/timezone');

dayjs.extend(utc);
dayjs.extend(timezone);

/**
 * 根据北京时间计算日期范围，返回 UTC Date 对象用于 MongoDB 查询
 * @param {string} range - 'today' | 'yesterday' | 'week' | 'month'
 * @returns {{ startDate: Date, endDate: Date }}
 */
function getDateRange(range) {
  const now = dayjs().tz('Asia/Shanghai');
  const todayStart = now.startOf('day');
  
  let start, end;
  switch (range) {
    case 'yesterday':
      start = todayStart.subtract(1, 'day');
      end = todayStart;
      break;
    case 'week':
      start = todayStart.startOf('week'); // 周一
      end = todayStart.add(1, 'day');
      break;
    case 'month':
      start = todayStart.startOf('month');
      end = todayStart.add(1, 'day');
      break;
    default: // today
      start = todayStart;
      end = todayStart.add(1, 'day');
  }
  
  return {
    startDate: start.toDate(),
    endDate: end.toDate(),
  };
}

/**
 * 获取最近 N 天的日期字符串数组（北京时间）
 * @param {number} days - 天数
 * @returns {string[]} ['2026-04-01', '2026-04-02', ...]
 */
function getRecentDates(days) {
  const now = dayjs().tz('Asia/Shanghai');
  const todayStart = now.startOf('day');
  
  const dates = [];
  for (let i = days - 1; i >= 0; i--) {
    dates.push(todayStart.subtract(i, 'day').format('YYYY-MM-DD'));
  }
  return dates;
}

module.exports = {
  getDateRange,
  getRecentDates,
};
```

### 2. IP 隐私保护优化

**方案 A：存储时脱敏（推荐）**

```javascript
// backend/src/utils/ipHelper.js
/**
 * 脱敏 IP 地址（保留前两段）
 * @param {string} ip - 原始 IP
 * @returns {string} 脱敏后的 IP（如 192.168.*.*）
 */
function maskIP(ip) {
  if (!ip) return '';
  
  // IPv4
  if (ip.includes('.')) {
    return ip.replace(/(\d+\.\d+)\.\d+\.\d+/, '$1.*.*');
  }
  
  // IPv6（保留前 4 段）
  if (ip.includes(':')) {
    const parts = ip.split(':');
    return parts.slice(0, 4).join(':') + ':****';
  }
  
  return ip;
}

module.exports = { maskIP };
```

```javascript
// backend/src/controllers/visitController.js
const { maskIP } = require('../utils/ipHelper');

exports.recordVisit = async (req, res) => {
  try {
    const path = req.body.path || req.body.url;
    const { sessionId } = req.body;

    if (!path || !sessionId) {
      return res.json({ code: 0, message: '记录成功' });
    }

    const rawIP = req.headers['x-forwarded-for']?.split(',')[0]?.trim()
      || req.headers['x-real-ip']
      || req.connection?.remoteAddress
      || '';

    await Visit.create({
      path,
      title: req.body.title || '',
      referer: req.body.referer || req.body.referrer || '',
      userAgent: req.headers['user-agent'] || '',
      ip: maskIP(rawIP), // 存储时就脱敏
      sessionId,
      duration: req.body.duration || null,
    });
  } catch {
    // 静默失败
  }
  res.json({ code: 0, message: '记录成功' });
};
```

**方案 B：哈希存储（更安全，但不可逆）**

```javascript
// backend/src/utils/ipHelper.js
const crypto = require('crypto');

const IP_SALT = process.env.IP_HASH_SALT || 'your-random-salt-here';

/**
 * 对 IP 进行单向哈希（用于 UV 统计，但无法还原）
 * @param {string} ip - 原始 IP
 * @returns {string} 哈希值
 */
function hashIP(ip) {
  if (!ip) return '';
  return crypto.createHash('sha256').update(ip + IP_SALT).digest('hex').substring(0, 16);
}

module.exports = { hashIP };
```

**推荐**：用方案 A（脱敏），既保护隐私，又能看到大致地理位置（前两段）。

### 3. 前端防抖逻辑优化

**问题**：`lastVisitTime` 存在内存中，页面刷新后丢失。

**解决**：将 `lastVisitTime` 也存到 `localStorage`。

```typescript
// frontend/src/utils/analytics.ts

class Analytics {
  private sessionId: string;
  private lastPath: string = '';
  private debounceTimer: ReturnType<typeof setTimeout> | null = null;
  private originalPushState: typeof history.pushState;
  private originalReplaceState: typeof history.replaceState;

  private readonly VISIT_INTERVAL = 5 * 60 * 1000; // 5 分钟
  private readonly DEBOUNCE_MS = 50;
  private readonly LAST_VISIT_KEY = 'last_visit_time';
  private readonly LAST_PATH_KEY = 'last_visit_path';

  constructor() {
    this.sessionId = this.getOrCreateSessionId();
    this.originalPushState = history.pushState.bind(history);
    this.originalReplaceState = history.replaceState.bind(history);
    
    // 从 localStorage 恢复上次访问信息
    this.lastPath = localStorage.getItem(this.LAST_PATH_KEY) || '';
  }

  private getOrCreateSessionId(): string {
    const key = 'visitor_session_id';
    let id = localStorage.getItem(key);
    if (!id) {
      id = `${Date.now()}_${Math.random().toString(36).substring(2, 15)}`;
      localStorage.setItem(key, id);
    }
    return id;
  }

  private getLastVisitTime(): number {
    return parseInt(localStorage.getItem(this.LAST_VISIT_KEY) || '0', 10);
  }

  private setLastVisitTime(time: number): void {
    localStorage.setItem(this.LAST_VISIT_KEY, time.toString());
  }

  public init(): void {
    this.trackPageView();

    window.addEventListener('popstate', this.handleRouteChange);

    history.pushState = (...args: Parameters<typeof history.pushState>) => {
      this.originalPushState(...args);
      this.trackPageView();
    };

    history.replaceState = (...args: Parameters<typeof history.replaceState>) => {
      const prev = location.pathname;
      this.originalReplaceState(...args);
      if (location.pathname !== prev) {
        this.trackPageView();
      }
    };
  }

  private handleRouteChange = (): void => {
    this.trackPageView();
  };

  public trackPageView(): void {
    if (this.debounceTimer !== null) {
      clearTimeout(this.debounceTimer);
    }
    this.debounceTimer = window.setTimeout(() => {
      this.doTrackPageView();
    }, this.DEBOUNCE_MS);
  }

  private doTrackPageView(): void {
    const currentPath = location.pathname;
    const now = Date.now();
    const lastTime = this.getLastVisitTime();

    // 同一页面 + 5 分钟内 → 不重复计数
    if (currentPath === this.lastPath && now - lastTime < this.VISIT_INTERVAL) {
      return;
    }

    this.lastPath = currentPath;
    this.setLastVisitTime(now);
    localStorage.setItem(this.LAST_PATH_KEY, currentPath);

    this.sendData({
      path: currentPath,
      title: document.title,
      referer: document.referrer,
      sessionId: this.sessionId,
    });
  }

  private sendData(data: Record<string, unknown>): void {
    try {
      const blob = new Blob([JSON.stringify(data)], { type: 'application/json' });
      if (!navigator.sendBeacon('/api/stats/visit', blob)) {
        fetch('/api/stats/visit', {
          method: 'POST',
          headers: { 'Content-Type': 'application/json' },
          body: JSON.stringify(data),
          keepalive: true,
        }).catch(() => {});
      }
    } catch {
      // 静默失败
    }
  }

  public destroy(): void {
    window.removeEventListener('popstate', this.handleRouteChange);
    history.pushState = this.originalPushState;
    history.replaceState = this.originalReplaceState;
    if (this.debounceTimer !== null) {
      clearTimeout(this.debounceTimer);
    }
  }

  public getSessionId(): string {
    return this.sessionId;
  }
}

const analytics = new Analytics();
export default analytics;
```

### 4. MongoDB 索引优化

**当前索引**：
```javascript
visitSchema.index({ createdAt: -1 }); // 降序
visitSchema.index({ path: 1 });
visitSchema.index({ sessionId: 1 });
visitSchema.index({ path: 1, createdAt: -1 });
```

**问题**：
- `getTrend` 查询 `{ createdAt: { $gte: startUTC } }` 时，降序索引效率不高
- `getOverview` 需要同时查询 `createdAt` 和 `sessionId`，但没有复合索引

**优化后**：
```javascript
// backend/src/models/Visit.js

// 删除旧索引，添加新索引
visitSchema.index({ createdAt: 1 }); // 改为升序，适合范围查询
visitSchema.index({ createdAt: 1, sessionId: 1 }); // 复合索引，覆盖 trend 查询
visitSchema.index({ path: 1, createdAt: -1 }); // 保留（热门页面查询）
visitSchema.index({ sessionId: 1 }); // 保留（UV 统计）
```

**迁移脚本**（生产环境执行）：
```javascript
// scripts/migrate-indexes.js
const mongoose = require('mongoose');
require('dotenv').config();

async function migrateIndexes() {
  await mongoose.connect(process.env.MONGO_URI);
  
  const Visit = mongoose.model('Visit');
  
  // 删除旧索引
  await Visit.collection.dropIndex('createdAt_-1');
  
  // 创建新索引
  await Visit.collection.createIndex({ createdAt: 1 });
  await Visit.collection.createIndex({ createdAt: 1, sessionId: 1 });
  
  console.log('索引迁移完成');
  process.exit(0);
}

migrateIndexes().catch(console.error);
```

### 5. 前端错误处理优化

**当前问题**：所有错误都显示"数据加载失败"。

**优化后**：
```typescript
// frontend/src/pages/admin/stats/index.tsx

const fetchData = useCallback(async () => {
  setStates({ 
    overview: 'loading', 
    trend: 'loading', 
    topPages: 'loading', 
    referers: 'loading', 
    visits: 'loading' 
  });

  const results = await Promise.allSettled([
    request('/api/stats/overview', { params: { range: timeRange } }),
    request('/api/stats/trend', { params: { days: trendDays } }),
    request('/api/stats/top-pages', { params: { limit: 10, range: timeRange } }),
    request('/api/stats/referers', { params: { limit: 5, range: timeRange } }),
    request('/api/stats/visits', { params: filter }),
  ]);

  const handlers: Array<{ region: string; setter: (data: any) => void }> = [
    { region: 'overview', setter: setOverview },
    { region: 'trend', setter: setTrend },
    { region: 'topPages', setter: setTopPages },
    { region: 'referers', setter: setReferers },
    { region: 'visits', setter: setVisits },
  ];

  results.forEach((r, i) => {
    if (r.status === 'fulfilled' && r.value?.code === 0) {
      handlers[i].setter(r.value.data);
      setRegion(handlers[i].region, 'success');
    } else {
      // 区分错误类型
      const error = r.status === 'rejected' ? r.reason : r.value;
      
      if (error?.response?.status === 401) {
        message.error('登录已过期，请重新登录');
        // 可选：跳转到登录页
        // history.push('/login');
      } else if (error?.response?.status === 403) {
        message.error('无权限访问统计数据');
      } else if (error?.response?.status === 500) {
        message.error('服务器错误，请稍后重试');
      } else if (!navigator.onLine) {
        message.error('网络连接已断开');
      } else {
        message.error(`${handlers[i].region} 数据加载失败`);
      }
      
      setRegion(handlers[i].region, 'error');
    }
  });
}, [timeRange, trendDays, filter, setRegion]);
```
