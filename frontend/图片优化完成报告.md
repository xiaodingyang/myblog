## 建议后续优化

### 针对首页 LCP 优化
1. **代码分割**: 减少首屏 JavaScript 体积
2. **懒加载**: 非首屏组件延迟加载
3. **Tree Shaking**: 移除未使用的代码
4. **CSS 优化**: 提取关键 CSS，延迟加载非关键 CSS
5. **字体优化**: 使用 `font-display: swap`

### 针对文章页面
1. **文章封面图优化**: 
   - 使用 CDN 加速
   - 配置响应式图片 (srcset)
   - 添加 `fetchpriority="high"` 给首屏图片
   - 启用图片懒加载

2. **服务器配置**:
   - 启用 HTTP/2
   - 配置 gzip/brotli 压缩
   - 设置合理的缓存策略
   - 使用 CDN
